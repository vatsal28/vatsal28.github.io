OpenClaw Agent

Cloidberg, an OpenClaw agent that lives on a mini PC at home. I talk to it on Telegram and Discord. It remembers what we're working on, helps me manage my life, home and side projects, and maintains my second brain.

Hermes Agent

Gilfoyle, my Hermes agent. This one looks after my home servers: it sets things up, watches what's running, and helps me repair things when they break.

Agent Memory & Second Brain

My knowledge management system to help me organize the chaos in my brain: markdown files holding notes, projects, sources, and ideas, all searchable instead of stuck in my head. Cloidberg keeps it organized, and it's the shared memory both my agents read from, so they don't start from zero every time and know what I care about.

Homelab

The actual servers in my house that run all of the above, along with a Wireguard VPN, a Plex media server, Nextcloud for files and photos, a Webmin dashboard so I can see what's going on, and some local AI models using Ollama. All on my own hardware.
